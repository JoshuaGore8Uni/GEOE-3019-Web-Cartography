About this site

This site showcases web maps I have created during the course "Web Cartography" and describes how each map was made. Most maps have been made using the Leaflet mapping library, but there are also examples of ArcGIS and Carto out of the box solutions, Mapbox GL JS (with custom styled maps), and Cesium. The capstone masterpiece of this site is a map showing the entire fire history of South Australia.

This site is hand coded using the Boostrap library and styled using the Flatly theme from Bootswatch. Most pages should work just fine on mobile, though the fire page needs to be viewed on a desktop.

Enjoy!
